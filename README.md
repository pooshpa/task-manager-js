# task-manager-js

## Description
This is a simple Task Manager application built using vanilla JavaScript. 
The project allows users to add, update, delete, and mark tasks as completed.
It is designed to strengthen core JavaScript and DOM manipulation skills.

## Features
- Add a new task
- Edit an existing task
- Delete a task
- Mark tasks as completed
- Save tasks using browser localStorage

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla JS)

## Learning Objectives
- Practice JavaScript fundamentals
- Understand DOM manipulation
- Work with browser localStorage
- Improve problem-solving skills
